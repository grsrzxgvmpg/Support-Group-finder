# Support Group Finder

An AI-powered application designed to help individuals find mental health support groups quickly and efficiently.

## Features
- **AI-Powered Search**: Uses Google Gemini API with Google Search grounding to find real, active support groups.
- **Smart URL Recovery**: Intelligently identifies verified direct links to organization websites.
- **Location Awareness**: Uses geolocation to find groups in your local community.
- **Detailed Group Views**: Modal-based detailed information including contact details, schedules, and source verification.
- **Responsive Design**: Calm, teal-themed UI optimized for mobile and desktop.

## Tech Stack
- **Frontend**: React (ES Modules)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **AI**: Google Gemini 2.5 Flash
- **Tools**: Google Search Grounding API
